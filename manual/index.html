Manual de instalação

1 - Crie a base de dados.

Abra a pasta _sql, nela há um dump para criar a base e a tabela necessária para a aplicação.

2 - Configurações.

Na raiz do projeto, abra em um editor o arquivo config.php.
Na linha 6 você definirá o diretório do projeto. No navegador o projeto deverá
ficar assim : http://localhost/jogodavelha.
Nas linhas 16,17,18 e 19 deverá informar os dados de conexão ao banco de dados criado previamente.

3 - Jogue.

Agora é só acessar a url da aplicação e jogar a vontade.
